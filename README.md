## Description
This program implements a menu-driven contact book application that allows users
to add, view, update, search, and delete contact details.

## Features
- Add new contacts
- Display all contacts
- Search contacts by mobile number or email
- Update existing contact details
- Delete contacts
- Input validation 

## Object-Oriented Design
- **Contact.cs**  
  Represents a single contact. All fields are private and accessed using
  public properties (encapsulation).

- **ContactBook.cs**  
  Manages a collection of contacts and provides methods for adding, searching,
  updating, and deleting contacts.

- **Program.cs**  
  Contains the main menu loop and handles user interaction.

## Data Structure
- `List<Contact>` is used to store contact objects dynamically.

## How to Run
1. Open the project in Visual Studio or JetBrains Rider
2. Build the solution
3. Run the application
4. Use the menu options displayed in the console

- C#
- .NET Console Application
